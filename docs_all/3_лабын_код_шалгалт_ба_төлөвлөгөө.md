- **Байршил:** `water_hub.html` мөр 135
- **Асуудал:** `url_for('water.standards')` дуудаж байгаа ч route handler байхгүй
- **Үр дагавар:** Хэрэглэгч "MNS Standard" карт дарахад 404 алдаа

#### W-3. Шинжилгээний форм — алдааны мэдэгдэл байхгүй (MEDIUM)
- **Байршил:** 4 analysis form template
- **Асуудал:** fetch() амжилтгүй бол хэрэглэгчид мэдэгдэхгүй
- **Засвар:** `.catch()` дотор toast/alert нэмэх

#### W-4. Давхар дээж нэмэх боломжтой (MEDIUM)
- **Байршил:** 4 analysis form — `addSample()` функц
- **Асуудал:** Сонгосон дээжийг dropdown-оос хасдаггүй
- **Засвар:** Сонгосон option-г disable эсвэл filter хийх

#### W-5. Дээжийн цаг хадгалагддаггүй (LOW)
- **Байршил:** `water_register.html` → `routes.py`
- **Асуудал:** `sample_time` form-оос ирж байгаа ч backend-д ашиглагддаггүй
