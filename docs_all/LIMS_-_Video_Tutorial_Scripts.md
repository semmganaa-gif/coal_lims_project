# LIMS - Video Tutorial Scripts

## Overview

Энэ баримт бичиг нь LIMS системийн хэрэглэгчийн сургалтын видео бичлэгийн төлөвлөгөө, скриптүүдийг агуулна.

---

## Video 1: Системд нэвтрэх ба үндсэн тойм (5 мин)

### Зорилго
Шинэ хэрэглэгч системд нэвтрэн, үндсэн интерфейстэй танилцах

### Script

```
[0:00] Танилцуулга
- "Сайн байна уу, LIMS системийн сургалтанд тавтай морилно уу"
- "Энэ видеонд бид системд хэрхэн нэвтрэх, үндсэн функцуудтай танилцах болно"

[0:30] Нэвтрэх хуудас
- Хөтөч нээх → lims.example.com
- Хэрэглэгчийн нэр, нууц үг оруулах
- "Нэвтрэх" товч дарах

[1:00] Dashboard тойм
- Үндсэн статистик (өнөөдрийн дээж, шинжилгээ)
- Хурдан холбоосууд
- Сүүлийн үйл ажиллагаа

[2:00] Цэсний бүтэц
- Дээж → Дээжийн жагсаалт, бүртгэл
- Шинжилгээ → Ажлын талбар, үр дүн
- Тайлан → Тайлан үүсгэх
- Чанар → QC хяналт, Westgard
- Тохиргоо → Хувийн тохиргоо

[3:30] Профайл тохиргоо
- Профайл зураг
- Нууц үг солих
- Мэдэгдлийн тохиргоо

[4:30] Дүгнэлт
- "Одоо та системийн үндсэн бүтэцтэй танилцлаа"
- "Дараагийн видеонд дээж бүртгэх талаар үзнэ"
```

### Demo Flow
1. Login page → Enter credentials → Click Login
2. Show dashboard widgets
3. Click through main menu items
4. Open profile settings

## Video 2: Дээж бүртгэх (8 мин)

### Зорилго
Шинэ дээж бүртгэх, шинжилгээ оноох

### Script

```
[0:00] Танилцуулга
- "Энэ видеонд дээж хэрхэн бүртгэх талаар үзнэ"

[0:30] Дээжийн жагсаалт
- Дээж → Жагсаалт
- Хайлт, шүүлтүүр ашиглах
- Багана нуух/харуулах

[2:00] Шинэ дээж үүсгэх
- "Шинэ дээж" товч дарах
- Дээжийн код оруулах (автомат/гараар)
- Захиалагч сонгох
- Дээжийн төрөл сонгох

[4:00] Шинжилгээ оноох
- Шинжилгээний жагсаалтаас сонгох (Aad, Mad, Vad, CV...)
- Бүлэг сонголт (Proximate, Ultimate)
- "Хадгалах" товч

[5:30] Барcode хэвлэх
- Дээжийн дэлгэрэнгүй → Barcode хэвлэх
- Label хэмжээ сонгох
- Хэвлэгч сонгох

[6:30] Дээж засах
- Дээжийн дэлгэрэнгүй → Засах
- Мэдээлэл өөрчлөх
- Шинжилгээ нэмэх/хасах

[7:30] Дүгнэлт
```

### Demo Data
- Sample code: QC-2026-001
- Client: QC
- Analyses: Aad, Mad, Vad, CV

## Video 3: Шинжилгээний үр дүн оруулах (10 мин)

### Зорилго
Ажлын талбар ашиглан шинжилгээний үр дүн оруулах

### Script

```
[0:00] Танилцуулга
- "Шинжилгээний үр дүн хэрхэн оруулах талаар үзнэ"

[0:30] Ажлын талбар
- Шинжилгээ → Ажлын талбар
- Шинжилгээний төрөл сонгох (Aad)
- Оноогдсон дээжүүд харагдана

[2:00] Үр дүн оруулах
- Дээж сонгох
- Түүхий өгөгдөл оруулах (A, B, C утгууд)
- Эцсийн үр дүн автоматаар тооцоолно

[4:00] Олон дээжид оруулах
- AG Grid ашиглан хурдан оруулах
- Copy/Paste ашиглах
- Tab товч ашиглах

[5:30] Автомат шалгалт
- Repeatability шалгалт
- CM/GBW хязгаар шалгалт
- Статус: approved, pending, rejected

[7:00] Засах, устгах
- Буруу утга засах
- Шинжилгээ устгах (зөвхөн админ)

[8:30] Хадгалах
- "Хадгалах" товч
- Амжилттай мессеж
- Audit log үүснэ

[9:30] Дүгнэлт
```

## Video 5.5: Лабораторийн модулиуд (8 мин)

### Зорилго
4 лабораторийн модулиуд хэрхэн ажилладаг, лаб сонгох, шилжих

### Script

```
[0:00] Танилцуулга
- "Энэ видеонд системийн 4 лабораторийн модулийг хэрхэн ашиглах талаар үзнэ"

[0:30] Лаб сонгох
- Dashboard дээрх лабын карт
- Нүүрсний лаб (Coal) — /labs/coal
- Усны лаб (Water) — /labs/water
- Микробиологи (Microbiology) — /labs/microbiology
- Петрограф (Petrography) — /labs/petrography

[2:00] Усны лабын жишээ
- Дээж бүртгэх (усны дээж)
- Шинжилгээ оноох (pH, EC, TDS гэх мэт)
- Workspace ашиглах
- Үр дүн хадгалах

[4:00] Микробиологийн лаб
- 3 ангиллын workspace (Ус, Агаар, Арчдас)
- Batch оруулах
- Үр дүн хянах

[5:30] Петрограф лаб
- PE дээж сонгох
- Мацерал, Витринит шинжилгээ
- Үр дүн хадгалах

[7:00] Лабын эрх (allowed_labs)
- Админ → бүх лабд хандах
- Хэрэглэгч → зөвхөн зөвшөөрөгдсөн лабд

[7:30] Дүгнэлт
```

### Demo Flow
1. Dashboard → Lab cards → Click Water Lab
2. Register water sample → Assign analyses
3. Switch to Microbiology → Show 3 workspace types
4. Switch to Petrography → Show PE sample analysis
